## Release v1.0.0

**First public release** — Peak Pro BLE integration for Home Assistant.

### Highlights

- Climate-based session control with profile presets and heat timer
- Profile temperature + duration (1–4), lantern RGB/effects, stealth mode
- Auto-reconnect, charging sensors, device automations, blueprints
- Flat + Lorax firmware support

### Install

**HACS:** Add custom repo → Install → Restart HA → Add integration.

**Manual:** Unzip `puffco-home-assistant-1.0.0.zip` into your config folder so you have `custom_components/puffco/`, then restart.

### Requirements

- Home Assistant 2024.6+
- Connectable Bluetooth or proxy
- Close Puffco phone app while using HA
